# Privacy Policy — YouTube Music DJ

Last updated: June 2025

## Data Collection
YouTube Music DJ does NOT collect any user data.

## What we access
- **Song title and artist name** on music.youtube.com: used only to generate 
  the voice announcement. This data is never stored or transmitted anywhere.
- **User preferences** (language, volume, template): stored locally in your 
  browser via chrome.storage.sync. Never sent to any server.

## Third-party services
This extension uses Google Translate TTS to generate audio. The song title 
and artist name are sent to Google's servers solely for text-to-speech 
conversion. Please refer to Google's Privacy Policy for more information.

## Contact
If you have any questions, please open an issue on this GitHub repository.
